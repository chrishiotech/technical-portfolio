# Technical Portfolio

Este repositorio contiene 5 proyectos técnicos que demuestran habilidades en Angular, React, React Native, AI Automation y DevOps.

## Proyectos

1. **Angular Enterprise Application** - Aplicación empresarial con Angular 17, NgRx, RxJS, Jest, Cypress, Vercel.
2. **Next.js MicroFrontend Demo** - Demostración de arquitectura MicroFrontend con Next.js 14, Webpack Module Federation, React, TypeScript, GitHub Actions.
3. **React Native Mobile App** - Aplicación móvil nativa con React Native, Expo, TypeScript, REST API.
4. **AI Automation Pipeline** - Pipeline de automatización con IA de bajo costo usando Python (FastAPI), HTMX, DeepSeek/Minimax API, Render.com.
5. **DevOps/IaC Infrastructure** - Infraestructura como código con Terraform/AWS CDK, GitHub Actions, Docker.

## Estructura

- `01-angular-enterprise/` - Proyecto 1
- `02-nextjs-microfrontend/` - Proyecto 2  
- `03-react-native-app/` - Proyecto 3
- `04-ai-automation/` - Proyecto 4
- `05-devops-iac/` - Proyecto 5

Cada proyecto tendrá su propio README con detalles de instalación, uso y demostración.

## Estado

Cada proyecto se desarrollará en el orden indicado, con un tiempo estimado de 1 semana por proyecto (10 horas/semana).
